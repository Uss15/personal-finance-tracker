# 💰 Personal Finance Tracker

A sleek and interactive web application built with Python and Streamlit to help users track their daily expenses, visualize spending habits, and manage their budget effectively.

## Features ✨
- **Interactive Dashboard**: Real-time summary metrics including total spent, average transaction, and total transactions.
- **Data Visualization**: Beautiful charts (Pie and Line) powered by Plotly to analyze spending by category and over time.
- **Expense Management**: Easily add new expenses with categories, descriptions, and dates.
- **Data Persistence**: Automatically saves your data to a local CSV file for future use.
- **Clean UI**: Modern and responsive design for a great user experience.

## Tech Stack 🛠️
- **Python**: Core programming language.
- **Streamlit**: Framework for building the web interface.
- **Pandas**: Data manipulation and analysis.
- **Plotly**: Interactive data visualization.

## Installation 🚀

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/personal-finance-tracker.git
   cd personal-finance-tracker
   ```

2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**:
   ```bash
   streamlit run app.py
   ```

## Usage 📱
- Use the sidebar to enter your expense details (Date, Category, Description, Amount).
- Click "Add Expense" to update the dashboard instantly.
- Explore the charts to see where your money is going.
- View the full transaction history in the table at the bottom.
